This is a seperate scroll wheel for making scrolling through datasheets and code more enjoyable. The idea is based on the Haptic Input Knob by scottbez1 on Youtube.
The microcontroller that I will use is an Xiao Seeed ESP32-C3 for its possibility of connecting through bluetooth and built-in battery charging for the used 3.7V 240mAh LiPo battery.

It is currently in a very early build, but I will keep this updated throughout the project.
